# Il fork perchè?
Questo fork è un semplice fix del port del layout `Italian US` fatto da [Sclafus](https://github.com/Sclafus), che soffriva (se non soffre ancora) del fatto di non essere capace di usare la dieresi e altri segni diacritici come segni combinanti: cioè era impossibile scrivere `e` e dopo `¨` per ottenere `ë` e quindi appariva `e¨`. Ad ogni modo, questa versione sembra essere una conversione fedele.
